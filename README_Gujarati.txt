**VS Code Auto GitHub Push System – Gujarati Guide**

1. સૌપ્રથમ `auto_git.sh` સ્ક્રિપ્ટ ને run કરવાની permission આપો:
   chmod +x auto_git.sh

2. પછી સ્ક્રિપ્ટ ચાલુ કરો:
   ./auto_git.sh

→ આ स्क्रિપ્ટ દર 60 secondsે આપમેળે commit અને push કરશે.

**VS Code Tasks થી Auto Start કરવું હોય તો:**
→ VS Code માં `Ctrl + Shift + B` દબાવો
→ “Auto Git Commit and Push” Task run કરો

Enjoy hassle-free GitHub auto push!
